# walk8243-css
個人用に作成したCSSのテンプレート

## GETTING STARTED
### npm module
#### install
```
npm i walk8243-css
```
#### Example
```example.js
const walk8243Css = require('walk8243-css');

walk8243Css();
```

### git clone
#### コマンド
- `npm run gulp`  
Gulpを起動し、Sassをコンパイルする。
- `npm run watch`  
Gulpを起動し、Sassファイルを監視し、変更されるとSassのコンパイルを実行する。

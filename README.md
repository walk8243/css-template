# @walk8243/css-template
個人用に作成したCSSのテンプレート

## GETTING STARTED
### npm module
#### install
```
# npm
npm i @walk8243/css-template

# yarn
yarn add @walk8243/css-template --update-checksums
```
#### Example
```example.js
const Walk8243Css = require('@walk8243/css-template');

Walk8243Css();
```
